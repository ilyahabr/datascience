Оценка тональности отзывов на товары

Инструкция по запуску:
1. Установите Flask (pip install Flask)
2. Распакуйте архив (final.zip)
3. Запустите сервер (python prog.py). По умолчанию сервер будет привязан на порту http://127.0.0.1:5000
4. Перейдите в браузере на http://127.0.0.1:5000/sentiment-demo
5. Введите текст описания (на русском языке)
6. Нажмите Оценить
